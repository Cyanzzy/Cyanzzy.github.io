з®—жі•еҜјеӯҰ-1-ж•°з»„зұ»еһӢйўҳзӣ®

дәҢеҲҶжҹҘжүҫ

LeetCode 704.дәҢеҲҶжҹҘжүҫ

й—®йўҳ

з»ҷе®ҡдёҖдёӘ n дёӘе…ғзҙ жңүеәҸзҡ„пјҲеҚҮеәҸпјүж•ҙеһӢж•°з»„ nums е’ҢдёҖдёӘзӣ®ж ҮеҖј target пјҢеҶҷдёҖдёӘеҮҪж•°жҗңзҙў nums дёӯзҡ„ targetпјҢеҰӮжһңзӣ®ж ҮеҖјеӯҳеңЁиҝ”еӣһдёӢж ҮпјҢеҗҰеҲҷиҝ”еӣһ -1гҖӮ

йўҳи§Ј

иҝҷйҒ“йўҳзӣ®зҡ„еүҚжҸҗжҳҜж•°з»„дёәжңүеәҸж•°з»„пјҢеҗҢж—¶йўҳзӣ®иҝҳејәи°ғж•°з»„дёӯж— йҮҚеӨҚе…ғзҙ пјҢеӣ дёәдёҖж—ҰжңүйҮҚеӨҚе…ғзҙ пјҢдҪҝз”ЁдәҢеҲҶжҹҘжүҫжі•иҝ”еӣһзҡ„е…ғзҙ дёӢж ҮеҸҜиғҪдёҚжҳҜе”ҜдёҖзҡ„пјҢиҝҷдәӣйғҪжҳҜдҪҝз”ЁдәҢеҲҶжі•зҡ„еүҚжҸҗжқЎд»¶

еҶҷдәҢеҲҶжі•пјҢеҢәй—ҙзҡ„е®ҡд№үдёҖиҲ¬дёәдёӨз§ҚпјҢе·Ұй—ӯеҸій—ӯеҚі[left, right]пјҢжҲ–иҖ…е·Ұй—ӯеҸіејҖеҚі[left, right)гҖӮ

е·Ұй—ӯеҸій—ӯзҡ„еҢәй—ҙ[left, right]
targetеңЁ[left, right]еҢәй—ҙпјҢжүҖд»ҘжңүеҰӮдёӢдёӨзӮ№пјҡ

  • while (left <= right) иҰҒдҪҝз”Ё <= пјҢеӣ дёәleft == rightжҳҜжңүж„Ҹд№үзҡ„пјҢжүҖд»ҘдҪҝз”Ё <=
  • if (nums[middle] > target) right иҰҒиөӢеҖјдёә middle - 1пјҢеӣ дёәеҪ“еүҚиҝҷдёӘnums[middle]дёҖе®ҡдёҚжҳҜtargetпјҢйӮЈд№ҲжҺҘдёӢжқҘиҰҒжҹҘжүҫзҡ„е·ҰеҢәй—ҙз»“жқҹдёӢж ҮдҪҚзҪ®е°ұжҳҜ middle - 1

е·Ұй—ӯеҸіејҖзҡ„еҢәй—ҙ[left, right)

  • while (left < right)пјҢиҝҷйҮҢдҪҝз”Ё <пјҢеӣ дёәleft == rightеңЁеҢәй—ҙ[left, right)жҳҜжІЎжңүж„Ҹд№үзҡ„
  • if (nums[middle] > target) right жӣҙж–°дёә middleпјҢ==еӣ дёәеҪ“еүҚnums[middle]дёҚзӯүдәҺtargetпјҢеҺ»е·ҰеҢәй—ҙз»§з»ӯеҜ»жүҫпјҢиҖҢеҜ»жүҫеҢәй—ҙжҳҜе·Ұй—ӯеҸіејҖеҢәй—ҙпјҢжүҖд»Ҙrightжӣҙж–°дёәmiddle==пјҢеҚіпјҡдёӢдёҖдёӘжҹҘиҜўеҢәй—ҙдёҚдјҡеҺ»жҜ”иҫғnums[middle]

еӨҚжқӮеәҰеҲҶжһҗ

з”ұдәҺжҜҸж¬ЎжҹҘжүҫйғҪдјҡе°ҶжҹҘжүҫиҢғеӣҙзј©е°ҸдёҖеҚҠпјҢеӣ жӯӨдәҢеҲҶжҹҘжүҫзҡ„ж—¶й—ҙеӨҚжқӮеәҰжҳҜ O(logn)пјҢе…¶дёӯ nжҳҜж•°з»„зҡ„й•ҝеәҰгҖӮ

з§»йҷӨе…ғзҙ 

LeetCode 27.з§»йҷӨе…ғзҙ 

й—®йўҳ

з»ҷдҪ дёҖдёӘж•°з»„ nums е’ҢдёҖдёӘеҖј valпјҢдҪ йңҖиҰҒ еҺҹең° з§»йҷӨжүҖжңүж•°еҖјзӯүдәҺ val зҡ„е…ғзҙ пјҢе№¶иҝ”еӣһз§»йҷӨеҗҺж•°з»„зҡ„ж–°й•ҝеәҰгҖӮ

дёҚиҰҒдҪҝз”ЁйўқеӨ–зҡ„ж•°з»„з©әй—ҙпјҢдҪ еҝ…йЎ»д»…дҪҝз”Ё O(1) йўқеӨ–з©әй—ҙе№¶ еҺҹең° дҝ®ж”№иҫ“е…Ҙж•°з»„гҖӮ

е…ғзҙ зҡ„йЎәеәҸеҸҜд»Ҙж”№еҸҳгҖӮдҪ дёҚйңҖиҰҒиҖғиҷ‘ж•°з»„дёӯи¶…еҮәж–°й•ҝеәҰеҗҺйқўзҡ„е…ғзҙ гҖӮ

еҸҢжҢҮй’Ҳжі•

йўҳи§Ј

дҪҝз”ЁеҸҢжҢҮй’ҲпјҢеҸіжҢҮй’ҲrightжҢҮеҗ‘еҪ“еүҚйңҖиҰҒеӨ„зҗҶзҡ„е…ғзҙ пјҢе·ҰжҢҮй’ҲleftжҢҮеҗ‘дёӢдёҖдёӘе°ҶиҰҒиөӢеҖјзҡ„дҪҚзҪ®

  • еҰӮжһңеҸіжҢҮй’ҲжҢҮеҗ‘зҡ„е…ғзҙ дёҚзӯүдәҺvalпјҢе®ғдёҖе®ҡжҳҜиҫ“еҮәж•°з»„зҡ„дёҖдёӘе…ғзҙ пјҢеҲҷе°ҶеҸіжҢҮй’ҲжҢҮеҗ‘зҡ„е…ғзҙ еӨҚеҲ¶еҲ°е·ҰжҢҮй’ҲдҪҚзҪ®пјҢз„¶еҗҺеҗҢж—¶з§»еҠЁе·ҰеҸіжҢҮй’Ҳ

  • еҰӮжһңеҸіжҢҮй’ҲжҢҮеҗ‘зҡ„е…ғзҙ зӯүдәҺvalпјҢе®ғдёҚиғҪеңЁиҫ“еҮәж•°з»„йҮҢпјҢжӯӨж—¶е·ҰжҢҮй’ҲдёҚеҠЁпјҢеҸіжҢҮй’ҲеҸіз§»дёҖдҪҚ

ж•ҙдёӘиҝҮзЁӢдҝқжҢҒдёҚеҸҳзҡ„жҖ§иҙЁжҳҜпјҡеҢәй—ҙ [0,left) дёӯзҡ„е…ғзҙ йғҪдёҚзӯүдәҺvalгҖӮеҪ“е·ҰеҸіжҢҮй’ҲйҒҚеҺҶе®Ңиҫ“е…Ҙж•°з»„д»ҘеҗҺпјҢleft зҡ„еҖје°ұжҳҜиҫ“еҮәж•°з»„зҡ„й•ҝеәҰгҖӮ

еӣҫи§Ј

еҲ йҷӨ2пјҢеҪ“е·ҰеҸіжҢҮй’ҲйғҪжҢҮеҗ‘0пјҲе·ҰжҢҮй’ҲжҢҮеҗ‘дёӢдёҖдёӘиҰҒиөӢеҖјзҡ„дҪҚзҪ®пјҢеҸіжҢҮй’ҲжҢҮеҗ‘еҪ“еүҚиҰҒеӨ„зҗҶзҡ„е…ғзҙ пјүпјҢжӯӨж—¶еҸіжҢҮй’Ҳзҡ„еҖјеӨҚеҲ¶еҲ°е·ҰжҢҮй’ҲдҪҚзҪ®пјҢе·ҰеҸіжҢҮй’ҲеҗҢж—¶еҸіз§»пјҢе·ҰеҸіжҢҮй’ҲжҢҮеҗ‘1еҗҢзҗҶгҖӮеҪ“е·ҰеҸіжҢҮй’ҲжҢҮеҗ‘еҲ°2пјҢжӯӨж—¶е·ҰжҢҮй’ҲдёҚеҠЁпјҢеҸіжҢҮй’ҲеҸіз§»гҖӮеҸіжҢҮй’ҲжқҘеҲ°3пјҢдёҚжҳҜ2пјҢеҸіжҢҮй’ҲеӨҚеҲ¶еҲ°е·ҰжҢҮй’ҲдҪҚзҪ®пјҢвҖҰ

еӨҚжқӮеәҰеҲҶжһҗ

O(n)пјҢе…¶дёӯ n дёәеәҸеҲ—зҡ„й•ҝеәҰгҖӮжҲ‘д»¬еҸӘйңҖиҰҒйҒҚеҺҶиҜҘеәҸеҲ—иҮіеӨҡдёӨж¬ЎпјҢ еңЁжңҖеқҸжғ…еҶөдёӢпјҲиҫ“е…Ҙж•°з»„дёӯжІЎжңүе…ғзҙ зӯүдәҺvalпјүпјҢе·ҰеҸіжҢҮй’Ҳеҗ„йҒҚеҺҶдәҶж•°з»„дёҖж¬ЎгҖӮ

еҸҢжҢҮй’ҲдјҳеҢ–жі•

йўҳи§Ј

еҰӮжһңиҰҒз§»йҷӨзҡ„е…ғзҙ жҒ°еҘҪеңЁж•°з»„зҡ„ејҖеӨҙпјҢдҫӢеҰӮеәҸеҲ— [1,2,3,4,5]пјҢеҪ“ val дёә 1 ж—¶пјҢжҲ‘д»¬йңҖиҰҒжҠҠжҜҸдёҖдёӘе…ғзҙ йғҪе·Ұз§»дёҖдҪҚгҖӮжіЁж„ҸеҲ°йўҳзӣ®дёӯиҜҙпјҡгҖҢе…ғзҙ зҡ„йЎәеәҸеҸҜд»Ҙж”№еҸҳгҖҚгҖӮе®һйҷ…дёҠжҲ‘д»¬еҸҜд»ҘзӣҙжҺҘе°ҶжңҖеҗҺдёҖдёӘе…ғзҙ  5 з§»еҠЁеҲ°еәҸеҲ—ејҖеӨҙпјҢеҸ–д»Је…ғзҙ  1пјҢеҫ—еҲ°еәҸеҲ— [5,2,3,4]пјҢеҗҢж ·ж»Ўи¶ійўҳзӣ®иҰҒжұӮгҖӮиҝҷдёӘдјҳеҢ–еңЁеәҸеҲ—дёӯval е…ғзҙ зҡ„ж•°йҮҸиҫғе°‘ж—¶йқһеёёжңүж•ҲгҖӮ

е®һзҺ°ж–№йқўпјҢжҲ‘д»¬дҫқз„¶дҪҝз”ЁеҸҢжҢҮй’ҲпјҢдёӨдёӘжҢҮй’ҲеҲқе§Ӣж—¶еҲҶеҲ«дҪҚдәҺж•°з»„зҡ„йҰ–е°ҫпјҢеҗ‘дёӯй—ҙз§»еҠЁйҒҚеҺҶиҜҘеәҸеҲ—гҖӮ

  • еҰӮжһңе·ҰжҢҮй’Ҳleft жҢҮеҗ‘зҡ„е…ғзҙ зӯүдәҺvalпјҢжӯӨж—¶е°ҶеҸіжҢҮй’Ҳright жҢҮеҗ‘зҡ„е…ғзҙ еӨҚеҲ¶еҲ°е·ҰжҢҮй’Ҳleft зҡ„дҪҚзҪ®пјҢз„¶еҗҺеҸіжҢҮй’Ҳ right е·Ұз§»дёҖдҪҚгҖӮ

  • еҰӮжһңиөӢеҖјиҝҮжқҘзҡ„е…ғзҙ жҒ°еҘҪд№ҹзӯүдәҺvalпјҢеҸҜд»Ҙз»§з»ӯжҠҠеҸіжҢҮй’Ҳright жҢҮеҗ‘зҡ„е…ғзҙ зҡ„еҖјиөӢеҖјиҝҮжқҘпјҲе·ҰжҢҮй’Ҳ left жҢҮеҗ‘зҡ„зӯүдәҺval зҡ„е…ғзҙ зҡ„дҪҚзҪ®з»§з»ӯиў«иҰҶзӣ–пјүпјҢзӣҙеҲ°е·ҰжҢҮй’ҲжҢҮеҗ‘зҡ„е…ғзҙ зҡ„еҖјдёҚзӯүдәҺ val дёәжӯўгҖӮ

  • еҪ“е·ҰжҢҮй’Ҳleft е’ҢеҸіжҢҮй’Ҳright йҮҚеҗҲзҡ„ж—¶еҖҷпјҢе·ҰеҸіжҢҮй’ҲйҒҚеҺҶе®Ңж•°з»„дёӯжүҖжңүзҡ„е…ғзҙ гҖӮ

  • ==дёҺж–№жі•дёҖдёҚеҗҢзҡ„жҳҜпјҢж–№жі•дәҢйҒҝе…ҚдәҶйңҖиҰҒдҝқз•ҷзҡ„е…ғзҙ зҡ„йҮҚеӨҚиөӢеҖјж“ҚдҪңгҖӮ==

жңүеәҸж•°з»„зҡ„е№іж–№

LeetCode 977.жңүеәҸж•°з»„зҡ„е№іж–№

й—®йўҳ

з»ҷдҪ дёҖдёӘжҢүйқһйҖ’еҮҸйЎәеәҸжҺ’еәҸзҡ„ж•ҙж•°ж•°з»„numsпјҢиҝ”еӣһжҜҸдёӘж•°еӯ—зҡ„е№іж–№з»„жҲҗзҡ„ж–°ж•°з»„пјҢиҰҒжұӮд№ҹжҢүйқһйҖ’еҮҸйЎәеәҸжҺ’еәҸгҖӮ

жҺ’еәҸ

з®—жі•жҖқжғі:

йЎәеәҸйҒҚеҺҶж•°з»„е№¶е®ҢжҲҗе№іж–№ж“ҚдҪңпјҢз„¶еҗҺзӣҙжҺҘжҺ’еәҸж“ҚдҪңпјҢжҺ’еәҸи°ғз”Ёзі»з»ҹAPIеҚ з”Ёж—¶й—ҙе’Ңз©әй—ҙгҖӮ

еҸҢжҢҮй’Ҳ+еҪ’е№¶жҺ’еәҸ

з®—жі•жҖқжғіпјҡ

  • еҰӮжһңж•°з»„ nums дёӯзҡ„жүҖжңүж•°йғҪжҳҜйқһиҙҹж•°пјҢйӮЈд№Ҳе°ҶжҜҸдёӘж•°е№іж–№еҗҺпјҢж•°з»„д»Қз„¶дҝқжҢҒеҚҮеәҸпјӣ
  • еҰӮжһңж•°з»„nums дёӯзҡ„жүҖжңүж•°йғҪжҳҜиҙҹж•°пјҢйӮЈд№Ҳе°ҶжҜҸдёӘж•°е№іж–№еҗҺпјҢж•°з»„дјҡдҝқжҢҒйҷҚеәҸгҖӮ
  • жүҫеҲ°ж•°з»„nums дёӯиҙҹж•°дёҺйқһиҙҹж•°зҡ„еҲҶз•ҢзәҝпјҢйӮЈд№Ҳе°ұеҸҜд»Ҙз”Ёзұ»дјј==гҖҢеҪ’е№¶жҺ’еәҸгҖҚ==зҡ„ж–№жі•дәҶгҖӮ
  • и®ҫnegдёәж•°з»„nums дёӯиҙҹж•°дёҺйқһиҙҹж•°зҡ„еҲҶз•ҢзәҝпјҢе°Ҷж•°з»„ nums дёӯзҡ„ж•°е№іж–№еҗҺпјҢйӮЈд№Ҳ nums[0] еҲ°nums[neg] еҚ•и°ғйҖ’еҮҸпјҢnums[neg+1] еҲ° nums[nвҲ’1] еҚ•и°ғйҖ’еўһгҖӮ
  • дҪҝз”ЁеҪ’е№¶зҡ„ж–№жі•иҝӣиЎҢжҺ’еәҸгҖӮдҪҝз”ЁдёӨдёӘжҢҮй’ҲеҲҶеҲ«жҢҮеҗ‘дҪҚзҪ® neg е’Ң neg+1пјҢжҜҸж¬ЎжҜ”иҫғдёӨдёӘжҢҮй’ҲеҜ№еә”зҡ„ж•°пјҢйҖүжӢ©иҫғе°Ҹзҡ„йӮЈдёӘж”ҫе…Ҙзӯ”жЎҲе№¶з§»еҠЁжҢҮй’ҲгҖӮеҪ“жҹҗдёҖжҢҮй’Ҳз§»иҮіиҫ№з•Ңж—¶пјҢе°ҶеҸҰдёҖжҢҮй’ҲиҝҳжңӘйҒҚеҺҶеҲ°зҡ„ж•°дҫқж¬Ўж”ҫе…Ҙзӯ”жЎҲгҖӮ

еӨҚжқӮеәҰеҲҶжһҗ

O(n)пјҢе…¶дёӯ n дёәж•°з»„й•ҝеәҰ

еҸҢжҢҮй’ҲдјҳеҢ–

дҪҝз”ЁеҸҢжҢҮй’ҲжҢҮеҗ‘йҰ–е°ҫпјҢжҜҸж¬ЎжҜ”иҫғдёӨдёӘжҢҮй’ҲеҜ№еә”зҡ„ж•°пјҢйҖүжӢ©иҫғеӨ§зҡ„йӮЈдёӘйҖҶеәҸж”ҫе…Ҙзӯ”жЎҲе№¶з§»еҠЁжҢҮй’ҲгҖӮиҝҷз§Қж–№жі•ж— йңҖеӨ„зҗҶжҹҗдёҖжҢҮй’Ҳз§»еҠЁиҮіиҫ№з•Ңзҡ„жғ…еҶө

й•ҝеәҰжңҖе°Ҹзҡ„еӯҗж•°з»„

LeetCode 209.й•ҝеәҰжңҖе°Ҹзҡ„еӯҗж•°з»„

й—®йўҳ

з»ҷе®ҡдёҖдёӘеҗ«жңүnдёӘжӯЈж•ҙж•°зҡ„ж•°з»„е’ҢдёҖдёӘжӯЈж•ҙж•°targetгҖӮ

жүҫеҮәиҜҘж•°з»„дёӯж»Ўи¶іе…¶е’ҢвүҘtargetзҡ„й•ҝеәҰжңҖе°Ҹзҡ„иҝһз»ӯеӯҗж•°з»„[numsl, numsl+1, ..., numsr-1, numsr]пјҢе№¶иҝ”еӣһе…¶й•ҝеәҰгҖӮеҰӮжһңдёҚеӯҳеңЁз¬ҰеҗҲжқЎд»¶зҡ„еӯҗж•°з»„пјҢиҝ”еӣһ0гҖӮ

жҡҙеҠӣжһҡдёҫ

з®—жі•жҖқжғіпјҡ

еҲқе§ӢеҢ–ж•°з»„жңҖе°Ҹй•ҝеәҰпјҢзҫҺеү§ж•°з»„numsдёӯзҡ„жҜҸдёӘдёӢж ҮдҪңдёәеӯҗж•°з»„зҡ„ејҖе§ӢдёӢж ҮпјҢеҜ№дәҺжҜҸдёӘејҖе§ӢдёӢж ҮiпјҢйңҖиҰҒжүҫеҲ°еӨ§дәҺзӯүдәҺiзҡ„жңҖе°ҸдёӢж ҮjпјҢдҪҝеҫ—д»Һnums[i]еҲ°nums[j]зҡ„е…ғзҙ е’ҢеӨ§дәҺзӯүдәҺsпјҢе№¶жӣҙж–°еӯҗж•°з»„зҡ„жңҖе°Ҹй•ҝеәҰпјҲжӯӨж—¶еӯҗж•°з»„зҡ„й•ҝеәҰжҳҜ$j-i+1$пјү

дјҡеҮәзҺ°и¶…ж—¶

еӨҚжқӮеәҰеҲҶжһҗ

O(n^2^)пјҢ е…¶дёӯ n жҳҜж•°з»„зҡ„й•ҝеәҰгҖӮйңҖиҰҒйҒҚеҺҶжҜҸдёӘдёӢж ҮдҪңдёәеӯҗж•°з»„зҡ„ејҖе§ӢдёӢж ҮпјҢеҜ№дәҺжҜҸдёӘејҖе§ӢдёӢж ҮпјҢйңҖиҰҒйҒҚеҺҶе…¶еҗҺйқўзҡ„дёӢж Үеҫ—еҲ°й•ҝеәҰжңҖе°Ҹзҡ„еӯҗж•°з»„гҖӮ

ж»‘еҠЁзӘ—еҸЈ

з®—жі•жҖқжғіпјҡ

е®ҡд№үдёӨдёӘжҢҮй’Ҳstart е’Ңend еҲҶеҲ«иЎЁзӨәеӯҗж•°з»„пјҲж»‘еҠЁзӘ—еҸЈзӘ—еҸЈпјүзҡ„ејҖе§ӢдҪҚзҪ®е’Ңз»“жқҹдҪҚзҪ®пјҢз»ҙжҠӨеҸҳйҮҸsum еӯҳеӮЁеӯҗж•°з»„дёӯзҡ„е…ғзҙ е’ҢпјҲеҚід»Һnums[start]еҲ°nums[end]зҡ„е…ғзҙ е’ҢпјүгҖӮ

  • еҲқе§ӢзҠ¶жҖҒдёӢпјҢstart е’Ңend йғҪжҢҮеҗ‘дёӢж Ү 0пјҢsum зҡ„еҖјдёә 0гҖӮ
  • жҜҸдёҖиҪ®иҝӯд»ЈпјҢе°Ҷnums[end]еҠ еҲ°sumпјҢеҰӮжһң$sumвүҘtarget$пјҢеҲҷжӣҙж–°еӯҗж•°з»„зҡ„жңҖе°Ҹй•ҝеәҰпјҲжӯӨж—¶еӯҗж•°з»„зҡ„й•ҝеәҰжҳҜ $endвҲ’start+1$пјүпјҢз„¶еҗҺе°Ҷnums[start]д»Һsum дёӯеҮҸеҺ»е№¶е°ҶstartеҸіз§»пјҢзӣҙеҲ° $sum<s$пјҢеңЁжӯӨиҝҮзЁӢдёӯеҗҢж ·жӣҙж–°еӯҗж•°з»„зҡ„жңҖе°Ҹй•ҝеәҰгҖӮ
  • еңЁжҜҸдёҖиҪ®иҝӯд»Јзҡ„жңҖеҗҺпјҢе°Ҷend еҸіз§»гҖӮ

    з®—жі•еӣҫи§Ј

ејҖе§Ӣж—¶пјҢйҰ–е°ҫжҢҮй’ҲжҢҮеҗ‘2пјҢsumдёә0

е°Ҷе°ҫжҢҮй’Ҳе…ғзҙ еҠ е…ҘпјҢsumжӣҙж–°дёә2

е°Ҷе°ҫжҢҮй’Ҳе…ғзҙ еҠ е…ҘпјҢsumжӣҙж–°дёә5

е°Ҷе°ҫжҢҮй’Ҳе…ғзҙ еҠ е…ҘпјҢsumжӣҙж–°дёә6

е°Ҷе°ҫжҢҮй’Ҳе…ғзҙ еҠ е…Ҙ,sumжӣҙж–°дёә8пјҢз”ұдәҺ8>targetпјҢans=4пјҢйҰ–жҢҮй’ҲеҸіз§»пјҢеү”йҷӨйҰ–е…ғзҙ 

ж»‘еҠЁзӘ—еҸЈеҶ…е…ғзҙ дёә3гҖҒ1гҖҒ2пјҢsumдёә6

е°Ҷе°ҫжҢҮй’Ҳе…ғзҙ еҠ е…ҘпјҢsumжӣҙж–°дёә10пјҢ10>targetпјҢans=4пјҢйҰ–жҢҮй’ҲеҸіз§»пјҢеү”йҷӨйҰ–е…ғзҙ 3

жӯӨж—¶sum=7пјҢ7=targetпјҢans=3пјҢйҰ–жҢҮй’ҲеҸіз§»пјҢеү”йҰ–йҷӨе…ғзҙ 1

жӯӨж—¶sum=6пјҢе°ҫжҢҮй’ҲеҮҶеӨҮеҸіз§»

е°Ҷе°ҫжҢҮй’Ҳе…ғзҙ еҠ е…ҘпјҢsumжӣҙж–°дёә9пјҢ9>targetпјҢans=3пјҢйҰ–жҢҮй’ҲеҸіз§»пјҢеү”йҷӨйҰ–е…ғзҙ 2

жӯӨж—¶sum=7пјҢ7=targetпјҢans=2пјҢйҰ–жҢҮй’ҲеҸіз§»пјҢеү”йҷӨйҰ–е…ғзҙ 

йҰ–е°ҫжҢҮй’ҲжқҘеҲ°ж•°з»„е°ҫйғЁпјҢдёҚж»Ўи¶іжқЎд»¶пјҢиҝ”еӣһans

еӨҚжқӮеәҰеҲҶжһҗ

O(n)пјҢе…¶дёӯ n жҳҜж•°з»„зҡ„й•ҝеәҰгҖӮжҢҮй’Ҳ start е’Ң end жңҖеӨҡеҗ„з§»еҠЁ n ж¬ЎгҖӮ

еүҚзјҖе’Ң+дәҢеҲҶжҹҘжүҫ

з®—жі•жҖқжғі

  • з”ұдәҺиҰҒжұӮиҝһз»ӯеӯҗж•°з»„пјҢеҸҜд»ҘеҲ©з”ЁеүҚзјҖе’ҢжҖқжғігҖӮйўқеӨ–еҲӣе»әдёҖдёӘж•°з»„sums з”ЁдәҺеӯҳеӮЁж•°з»„ nums зҡ„еүҚзјҖе’ҢпјҢе…¶дёӯsums[i]иЎЁзӨәд»Һnums[0]еҲ° nums[iвҲ’1]зҡ„е…ғзҙ е’Ң
  • еҫ—еҲ°еүҚзјҖе’Ңд№ӢеҗҺпјҢеҜ№дәҺжҜҸдёӘејҖе§ӢдёӢж ҮiпјҢеҸҜйҖҡиҝҮдәҢеҲҶжҹҘжүҫеҫ—еҲ°еӨ§дәҺжҲ–зӯүдәҺi зҡ„жңҖе°ҸдёӢж ҮboundпјҢдҪҝеҫ— $sums[bound]вҲ’sums[iвҲ’1]вүҘtarget$пјҢе№¶жӣҙж–°еӯҗж•°з»„зҡ„жңҖе°Ҹй•ҝеәҰпјҲжӯӨж—¶еӯҗж•°з»„зҡ„й•ҝеәҰжҳҜboundвҲ’(iвҲ’1)пјү
  • иҝҷйҒ“йўҳдҝқиҜҒдәҶж•°з»„дёӯжҜҸдёӘе…ғзҙ йғҪдёәжӯЈпјҢжүҖд»ҘеүҚзјҖе’ҢдёҖе®ҡжҳҜйҖ’еўһзҡ„пјҢиҝҷдёҖзӮ№дҝқиҜҒдәҶдәҢеҲҶзҡ„жӯЈзЎ®жҖ§гҖӮеҰӮжһңйўҳзӣ®жІЎжңүиҜҙжҳҺж•°з»„дёӯжҜҸдёӘе…ғзҙ йғҪдёәжӯЈпјҢиҝҷйҮҢе°ұдёҚиғҪдҪҝз”ЁдәҢеҲҶжқҘжҹҘжүҫиҝҷдёӘдҪҚзҪ®дәҶгҖӮ

дәҢеҲҶжҹҘжүҫеӨ§дәҺзӯүдәҺжҹҗж•°зҡ„з¬¬дёҖдёӘдёәдҪҚзҪ®

еӨҚжқӮеәҰеҲҶжһҗ

йңҖиҰҒйҒҚеҺҶжҜҸдёӘдёӢж ҮдҪңдёәеӯҗж•°з»„зҡ„ејҖе§ӢдёӢж ҮпјҢйҒҚеҺҶзҡ„ж—¶й—ҙеӨҚжқӮеәҰжҳҜ$O(n)$пјҢеҜ№дәҺжҜҸдёӘејҖе§ӢдёӢж ҮпјҢйңҖиҰҒйҖҡиҝҮдәҢеҲҶжҹҘжүҫеҫ—еҲ°й•ҝеәҰжңҖе°Ҹзҡ„еӯҗж•°з»„пјҢдәҢеҲҶжҹҘжүҫеҫ—ж—¶й—ҙеӨҚжқӮеәҰжҳҜ$O(logn)$пјҢеӣ жӯӨжҖ»ж—¶й—ҙеӨҚжқӮеәҰжҳҜ $O(nlogn)$гҖӮ

иһәж—Ӣзҹ©йҳө II

LeetCode 59. иһәж—Ӣзҹ©йҳө II

й—®йўҳ

з»ҷдҪ дёҖдёӘжӯЈж•ҙж•° n пјҢз”ҹжҲҗдёҖдёӘеҢ…еҗ« 1 еҲ° n2 жүҖжңүе…ғзҙ пјҢдё”е…ғзҙ жҢүйЎәж—¶й’ҲйЎәеәҸиһәж—ӢжҺ’еҲ—зҡ„ n x n жӯЈж–№еҪўзҹ©йҳө matrix гҖӮ

е…ЁеұҖи§’еәҰ

еңЁзҹ©йҳөдёӯз”Ёе·ҰдёҠи§’зҡ„еқҗж Ү$(tR,tC)$е’ҢеҸідёӢи§’еқҗж Ү$(dR,dC)$еҸҜд»ҘиЎЁзӨәдёҖдёӘеӯҗзҹ©йҳө

еҪ“$(tR,tC)=(0,0)$пјҢ$(dR,dC)=(3,3)$ж—¶пјҢиЎЁзӨәзҡ„еӯҗзҹ©йҳөе°ұжҳҜж•ҙдёӘзҹ©йҳөпјҢзҹ©йҳөжңҖеӨ–еұӮеҰӮдёҠпјҢе…ҲжҠҠжңҖеӨ–еұӮзҹ©йҳөжү“еҚ°пјҢз„¶еҗҺtR++пјҢtC++пјҢdR--пјҢdC--иЎЁзӨәзҡ„еӯҗзҹ©йҳөеҰӮдёӢпјҡ


еҶҚжҠҠеҶ…еұӮзҹ©йҳөжү“еҚ°еҮәжқҘеҚіеҸҜпјҢеҰӮжһңеҸ‘зҺ°е·ҰдёҠи§’еқҗж Үи·‘еҲ°дәҶеҸідёӢи§’еқҗж Үзҡ„еҸіж–№жҲ–дёӢж–№пјҢж•ҙдёӘиҝҮзЁӢеҒңжӯўгҖӮ

еӨҚжқӮеәҰеҲҶжһҗ

$O(n^2)$пјҢе…¶дёӯ n жҳҜж•°з»„зҡ„й•ҝеәҰ

еұҖйғЁи§’еәҰ

ж•°з»„зұ»еһӢжҖ»з»“

  • дәҢеҲҶжҹҘжүҫеҸӘйҖӮз”ЁдәҺжңүеәҸеәҸеҲ—е№¶дё”ж— йҮҚеӨҚе…ғзҙ пјҢеӣ дёәдёҖж—ҰжңүйҮҚеӨҚе…ғзҙ пјҢдҪҝз”ЁдәҢеҲҶжҹҘжүҫжі•иҝ”еӣһзҡ„е…ғзҙ дёӢж ҮеҸҜиғҪдёҚжҳҜе”ҜдёҖзҡ„пјҢиҝҷдәӣйғҪжҳҜдҪҝз”ЁдәҢеҲҶжі•зҡ„еүҚжҸҗжқЎд»¶
  • еҸҢжҢҮй’ҲжҖқжғіпјҢеҰӮLeetCode 977.жңүеәҸж•°з»„зҡ„е№іж–№
  • ж»‘еҠЁзӘ—еҸЈжҖқжғіпјҢеҰӮLeetCode 209.й•ҝеәҰжңҖе°Ҹзҡ„еӯҗж•°з»„
  • еүҚзјҖе’ҢжҖқжғіпјҢеҰӮLeetCode 209.й•ҝеәҰжңҖе°Ҹзҡ„еӯҗж•°з»„
  • жЁЎжӢҹзұ»еһӢпјҢеҸҜд»ҘжҢүеұӮж•ҙдҪ“еӨ„зҗҶпјҢд№ҹеҸҜд»ҘеұҖйғЁеӨ„зҗҶпјҢеҰӮLeetCode 59. иһәж—Ӣзҹ©йҳө II
