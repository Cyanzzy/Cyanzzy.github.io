Nacosдёӯзҡ„dataidзҡ„з»„жҲҗж јејҸеҸҠдёҺSpringBootй…ҚзҪ®ж–Үд»¶дёӯзҡ„еҢ№й…Қи§„еҲҷ

еңЁ Nacos Spring Cloud дёӯпјҢdataId зҡ„е®Ңж•ҙж јејҸеҰӮдёӢпјҡ

  • prefix й»ҳи®Өдёә spring.application.name зҡ„еҖјпјҢд№ҹеҸҜд»ҘйҖҡиҝҮй…ҚзҪ®йЎ№ spring.cloud.nacos.config.prefixжқҘй…ҚзҪ®гҖӮ
  • spring.profiles.active еҚідёәеҪ“еүҚзҺҜеўғеҜ№еә”зҡ„ profileпјҢиҜҰжғ…еҸҜд»ҘеҸӮиҖғ Spring Bootж–ҮжЎЈгҖӮ жіЁж„ҸпјҡеҪ“ spring.profiles.active дёәз©әж—¶пјҢеҜ№еә”зҡ„иҝһжҺҘз¬Ұ - д№ҹе°ҶдёҚеӯҳеңЁпјҢdataId зҡ„жӢјжҺҘж јејҸеҸҳжҲҗ ${prefix}.${file-extension}
  • file-exetension дёәй…ҚзҪ®еҶ…е®№зҡ„ж•°жҚ®ж јејҸпјҢеҸҜд»ҘйҖҡиҝҮй…ҚзҪ®йЎ№ spring.cloud.nacos.config.file-extension жқҘй…ҚзҪ®гҖӮзӣ®еүҚеҸӘж”ҜжҢҒ properties е’Ң yaml зұ»еһӢгҖӮ

  • prodз”ҹдә§зҺҜеўғ

еҰӮдҪ•дҝқиҜҒжҢҮе®ҡзҺҜеўғеҗҜеҠЁж—¶жңҚеҠЎиғҪжӯЈзЎ®иҜ»еҸ–еҲ°Nacosзӣёеә”зҺҜеўғзҡ„й…ҚзҪ®ж–Үд»¶пјҹ

й—®йўҳ2

дёҖдёӘеӨ§еһӢеҲҶеёғејҸеҫ®жңҚеҠЎзі»з»ҹдјҡжңүеҫҲеӨҡеҫ®жңҚеҠЎеӯҗйЎ№зӣ®пјҢжҜҸдёӘеҫ®жңҚеҠЎйЎ№зӣ®еҸҲдјҡжңүзӣёеә”зҡ„ејҖеҸ‘зҺҜеўғгҖҒжөӢиҜ•зҺҜеўғгҖҒйў„еҸ‘зҺҜеўғгҖҒжӯЈејҸзҺҜеўғзӯүпјҢ

еҰӮдҪ•еҜ№иҝҷдәӣеҫ®жңҚеҠЎй…ҚзҪ®иҝӣиЎҢз®ЎзҗҶпјҹ

Namespace+Group+DataID

1.жҳҜд»Җд№Ҳ

зұ»дјјJavaйҮҢйқўзҡ„еҢ…еҗҚе’Ңзұ»еҗҚпјҢжңҖеӨ–еұӮзҡ„namespaceеҸҜд»Ҙз”ЁдәҺеҢәеҲҶйғЁзҪІзҺҜеўғпјҢGroupе’ҢDataIDйҖ»иҫ‘дёҠеҢәеҲҶдёӨдёӘзӣ®ж ҮеҜ№иұЎ

2.дёүз§Қжғ…еҶө

й»ҳи®Өжғ…еҶө

Namespace=public, Group=DEFAULT_GROUPпјҢй»ҳи®ӨClusterжҳҜDEFAULT

  • Nacosй»ҳи®Өзҡ„е‘ҪеҗҚз©әй—ҙжҳҜpublicпјҢNamespaceдё»иҰҒз”ЁдәҺе®һзҺ°йҡ”зҰ»
  • Groupй»ҳи®ӨжҳҜDEFAULT_GROUPпјҢGroupеҸҜд»ҘжҠҠдёҚеҗҢзҡ„еҫ®жңҚеҠЎеҲ’еҲҶеҲ°еҗҢдёҖдёӘеҲҶз»„йҮҢйқўеҺ»

  • Serviceе°ұжҳҜеҫ®жңҚеҠЎпјҢдёҖдёӘServiceеҸҜд»ҘеҢ…еҗ«еӨҡдёӘClusterпјҲйӣҶзҫӨпјүпјҢNacosй»ҳи®ӨClusterжҳҜDEFAULTпјҢClusterжҳҜеҜ№жҢҮе®ҡеҫ®жңҚеҠЎзҡ„дёҖдёӘиҷҡжӢҹеҲ’еҲҶ

  • Instanceе°ұжҳҜеҫ®жңҚеҠЎе®һдҫӢ

DataIDж–№жЎҲй…ҚзҪ®

жҢҮе®ҡspring.profiile.activeе’Ңй…ҚзҪ®ж–Үд»¶зҡ„DataIDжқҘдҪҝдёҚеҗҢзҺҜеўғдёӢиҜ»еҸ–дёҚеҗҢзҡ„й…ҚзҪ®

й»ҳи®Өз©әй—ҙ+й»ҳи®ӨеҲҶз»„+ж–°е»әdevе’ҢtestдёӨдёӘDataID

йҖҡиҝҮGroupе®һзҺ°зҺҜеўғеҢәеҲҶпјҢж–°е»әGroupпјҢеңЁNacosеӣҫеҪўз•ҢйқўжҺ§еҲ¶еҸ°ж–°е»әй…ҚзҪ®ж–Үд»¶
