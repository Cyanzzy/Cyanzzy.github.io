4 RestAPI

Created2023-07-24|Updated2023-09-15|Technology
|Word count:1.8k|Reading time:8min|Post View:

ESе®ҳж–№жҸҗдҫӣдәҶеҗ„з§ҚдёҚеҗҢиҜӯиЁҖзҡ„е®ўжҲ·з«ҜпјҢз”ЁжқҘж“ҚдҪңESгҖӮиҝҷдәӣе®ўжҲ·з«Ҝзҡ„жң¬иҙЁе°ұжҳҜз»„иЈ…DSLиҜӯеҸҘпјҢйҖҡиҝҮhttpиҜ·жұӮеҸ‘йҖҒз»ҷESгҖӮе®ҳж–№ж–ҮжЎЈең°еқҖпјҡhttps://www.elastic.co/guide/en/elasticsearch/client/index.html

е…¶дёӯзҡ„Java Rest ClientеҸҲеҢ…жӢ¬дёӨз§Қпјҡ

  • Java Low Level Rest Client
  • Java High Level Rest Client

mapping жҳ е°„еҲҶжһҗ

еҲӣе»әзҙўеј•еә“пјҢжңҖе…ій”®зҡ„жҳҜmappingжҳ е°„пјҢиҖҢmappingжҳ е°„иҰҒиҖғиҷ‘зҡ„дҝЎжҒҜеҢ…жӢ¬пјҡ

  • еӯ—ж®өеҗҚпјҡеҸҜд»ҘеҸӮиҖғж•°жҚ®иЎЁз»“жһ„зҡ„еҗҚз§°е’Ңзұ»еһӢ
  • еӯ—ж®өж•°жҚ®зұ»еһӢпјҡеҸҜд»ҘеҸӮиҖғж•°жҚ®иЎЁз»“жһ„зҡ„еҗҚз§°е’Ңзұ»еһӢ
  • жҳҜеҗҰеҸӮдёҺжҗңзҙўпјҡжҳҜеҗҰеҸӮдёҺжҗңзҙўиҰҒеҲҶжһҗдёҡеҠЎжқҘеҲӨж–ӯ
  • жҳҜеҗҰйңҖиҰҒеҲҶиҜҚпјҡеҶ…е®№еҰӮжһңжҳҜдёҖдёӘж•ҙдҪ“е°ұж— йңҖеҲҶиҜҚпјҢеҸҚд№ӢеҲҷиҰҒеҲҶиҜҚпјҢеҲҶиҜҚеҷЁпјҢжҲ‘д»¬еҸҜд»Ҙз»ҹдёҖдҪҝз”Ёik_max_word

й…’еә—ж•°жҚ®зҡ„зҙўеј•еә“з»“жһ„:

зү№ж®Ҡеӯ—ж®өиҜҙжҳҺпјҡ

  • locationпјҡең°зҗҶеқҗж ҮпјҢйҮҢйқўеҢ…еҗ«зІҫеәҰгҖҒзә¬еәҰ
  • allпјҡдёҖдёӘз»„еҗҲеӯ—ж®өпјҢе…¶зӣ®зҡ„жҳҜе°ҶеӨҡеӯ—ж®өзҡ„еҖј еҲ©з”Ёcopy_toеҗҲе№¶пјҢжҸҗдҫӣз»ҷз”ЁжҲ·жҗңзҙў

ең°зҗҶеқҗж ҮиҜҙжҳҺпјҡ

copy_toиҜҙжҳҺпјҡпјҲж”ҜжҢҒеӨҡдёӘеӯ—ж®өжҗңзҙўпјү

еҲқе§ӢеҢ– RestClient

еңЁelasticsearchжҸҗдҫӣзҡ„APIдёӯпјҢдёҺelasticsearchдёҖеҲҮдәӨдә’йғҪе°ҒиЈ…еңЁдёҖдёӘеҗҚдёәRestHighLevelClientзҡ„зұ»дёӯпјҢеҝ…йЎ»е…Ҳе®ҢжҲҗиҝҷдёӘеҜ№иұЎзҡ„еҲқе§ӢеҢ–пјҢе»әз«ӢдёҺelasticsearchзҡ„иҝһжҺҘгҖӮ

1.еј•е…Ҙesзҡ„RestHighLevelClientдҫқиө–пјҡ

2.еӣ дёәSpringBootй»ҳи®Өзҡ„ESзүҲжң¬жҳҜ7.6.2пјҢжүҖд»ҘжҲ‘д»¬йңҖиҰҒиҰҶзӣ–й»ҳи®Өзҡ„ESзүҲжң¬пјҡ

3.еҲқе§ӢеҢ–RestHighLevelClientпјҡ

иҝҷйҮҢдёәдәҶеҚ•е…ғжөӢиҜ•ж–№дҫҝпјҢжҲ‘д»¬еҲӣе»әдёҖдёӘжөӢиҜ•зұ»HotelIndexTestпјҢз„¶еҗҺе°ҶеҲқе§ӢеҢ–зҡ„д»Јз Ғзј–еҶҷеңЁ@BeforeEachж–№жі•дёӯпјҡ

еҲӣе»әзҙўеј•еә“

еҲӣе»әзҙўеј•еә“зҡ„APIеҰӮдёӢпјҡ

  • еҲӣе»әRequestеҜ№иұЎгҖӮеӣ дёәжҳҜеҲӣе»әзҙўеј•еә“зҡ„ж“ҚдҪңпјҢеӣ жӯӨRequestжҳҜCreateIndexRequestгҖӮ
  • ж·»еҠ иҜ·жұӮеҸӮж•°пјҢе…¶е®һе°ұжҳҜDSLзҡ„JSONеҸӮж•°йғЁеҲҶгҖӮеӣ дёәjsonеӯ—з¬ҰдёІеҫҲй•ҝпјҢиҝҷйҮҢжҳҜе®ҡд№үдәҶйқҷжҖҒеӯ—з¬ҰдёІеёёйҮҸMAPPING_TEMPLATEпјҢи®©д»Јз ҒзңӢиө·жқҘжӣҙеҠ дјҳйӣ…гҖӮ
  • еҸ‘йҖҒиҜ·жұӮпјҢclient.indices()ж–№жі•зҡ„иҝ”еӣһеҖјжҳҜIndicesClientзұ»еһӢпјҢе°ҒиЈ…дәҶжүҖжңүдёҺзҙўеј•еә“ж“ҚдҪңжңүе…ізҡ„ж–№жі•гҖӮ

е®ҡд№үmappingжҳ е°„зҡ„JSONеӯ—з¬ҰдёІеёёйҮҸпјҡ

зј–еҶҷеҚ•е…ғжөӢиҜ•пјҢе®һзҺ°еҲӣе»әзҙўеј•пјҡ

еҲ йҷӨзҙўеј•еә“

еҲ йҷӨзҙўеј•еә“зҡ„DSLиҜӯеҸҘйқһеёёз®ҖеҚ•пјҡ

дёҺеҲӣе»әзҙўеј•еә“зӣёжҜ”пјҡ

  • иҜ·жұӮж–№ејҸд»ҺPUTеҸҳдёәDELTE
  • иҜ·жұӮи·Ҝеҫ„дёҚеҸҳ
  • ж— иҜ·жұӮеҸӮж•°

дёүйғЁжӣІ

  • еҲӣе»әRequestеҜ№иұЎгҖӮиҝҷж¬ЎжҳҜDeleteIndexRequestеҜ№иұЎ
  • еҮҶеӨҮеҸӮж•°гҖӮиҝҷйҮҢжҳҜж— еҸӮ
  • 3пјүеҸ‘йҖҒиҜ·жұӮгҖӮж”№з”Ёdeleteж–№жі•

зј–еҶҷеҚ•е…ғжөӢиҜ•пјҢе®һзҺ°еҲ йҷӨзҙўеј•пјҡ

еҲӨж–ӯзҙўеј•еә“жҳҜеҗҰеӯҳеңЁ

еҲӨж–ӯзҙўеј•еә“жҳҜеҗҰеӯҳеңЁпјҢжң¬иҙЁе°ұжҳҜжҹҘиҜўпјҢеҜ№еә”зҡ„DSLжҳҜпјҡ

дёүйғЁжӣІ

  • еҲӣе»әRequestеҜ№иұЎгҖӮиҝҷж¬ЎжҳҜGetIndexRequestеҜ№иұЎ
  • еҮҶеӨҮеҸӮж•°гҖӮиҝҷйҮҢжҳҜж— еҸӮ
  • еҸ‘йҖҒиҜ·жұӮгҖӮж”№з”Ёexistsж–№жі•

жҖ»з»“

JavaRestClientж“ҚдҪңelasticsearchзҡ„жөҒзЁӢеҹәжң¬зұ»дјјгҖӮж ёеҝғжҳҜclient.indices()ж–№жі•жқҘиҺ·еҸ–зҙўеј•еә“зҡ„ж“ҚдҪңеҜ№иұЎгҖӮ

зҙўеј•еә“ж“ҚдҪңзҡ„еҹәжң¬жӯҘйӘӨпјҡ

  • еҲқе§ӢеҢ–RestHighLevelClient
  • еҲӣе»әXxxIndexRequestгҖӮXXXжҳҜCreateгҖҒGetгҖҒDelete
  • еҮҶеӨҮDSLпјҲ Createж—¶йңҖиҰҒпјҢе…¶е®ғжҳҜж— еҸӮпјү
  • еҸ‘йҖҒиҜ·жұӮгҖӮи°ғз”ЁRestHighLevelClient#indices().xxx()ж–№жі•пјҢxxxжҳҜcreateгҖҒexistsгҖҒdelete
